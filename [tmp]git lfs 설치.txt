* 사용 이유: github에 100MB 이상 파일 업로드
* 참고 블로그: https://jw910911.tistory.com/93

* 방법1
1) git lfs install -> 'lfs' is not a git command. 오류 뜨면 방법2 사용
2) git lfs track [추적할 파일]
3) 평소와 동일하게 git add, commit, push 진행하면 됨

* 방법2
1) url -s https://packagecloud.io/install/repositories/github/git-lfs/script.deb.sh | sudo bash
2) sudo apt-get install git-lfs
3) 방법1과 동일

* 기타 정보
- .h5 파일이 236MB 차지함
- 학습 데이터가 클 수록 커짐
